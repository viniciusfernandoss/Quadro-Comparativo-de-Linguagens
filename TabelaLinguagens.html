Quadro comparativo de linguagens

Linguagens Criador Ano de Lançamento Aplicação Descrição
JavaScript Brendan Eich 1995 Desenvolvimento Web JavaScript foi uma das principais linguagens de programação ao lado de HTML e CSS que ajudaram a construir a internet.
Java James Gosling 1995 Desenvolvimento de Aplicativos Corporativos Java continua sendo a linguagem de programação para a construção de aplicativos de nível empresarial há mais de 20 anos.
PHP Rasmus Lerdorf 1994 Desenvolvimento Web Hoje, o PHP tornou-se uma linguagem de script de uso geral que está sendo usada em todo o mundo, principalmente para desenvolvimento web.
Python Guido van Rossum 1990 Inteligência Artificial e Aprendizado de Máquina Destaca se nas tecnologias como Inteligência Artificial, Machine Learning e Data Analytics.
C# Anders Hejlsberg 1998 Desenvolvimento de aplicativos e Web usando .NET C# foi a abordagem da Microsoft para desenvolver uma linguagem de programação semelhante ao C orientado a objetos como parte de sua iniciativa .NET
R Ross Ihaka e Robert Gentleman 1993 Análise de Dados R se tornou uma das melhores linguagens de programação para projetos que exigem extensa análise de dados, modelagem gráfica de dados, análise espacial e de séries temporais.
